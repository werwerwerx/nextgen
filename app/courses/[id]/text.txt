1. Разработка основного функционала:

   - Автоматический парсер курсов
   - Автоматический запуск каждые 12 часов
   - Система отслеживания ошибок парсера
   - Отправка отчетов в админ-панель

   - Интеграция Meta Pixel (требуется настройка)

   - Расширенные метатеги OpenGraph
   - Favicon и логотип сайта
   - Оптимизация загрузки страниц
   - Настроенные sitemap и robots.txt

4. Административная панель (доступна по адресу /admin):
   - Система управления уведомлениями:

   - Управление курсами:
     • Включение/отключение курсов
     • Мониторинг отчетов о парсинге
     • Ручное редактирование информации
   - Управление заявками:
     • Просмотр статистики
     • Экспорт данных в CSV
     • Пагинация и фильтрация

Для завершения настройки сайта nextgen.appletownworld.com необходимо:

1. Настроить домен на Hostinger:
   - Тип: CNAME
   - Имя: nextgen
   - Значение: cname.vercel-dns.com
   - TTL: 3600

   После добавления DNS-записи Vercel автоматически проверит и подтвердит настройку домена. Процесс может занять до 24 часов, но обычно завершается в течение нескольких минут. Проверить можно на https://vercel.com/nextgen-9c28d5d1/nextgen/settings/domains - с домена должно уйти сообщение о не корректной настройке

2. Настроить API ключи:
   - Настройка Supabase:
      1. Зайдите в Supabase Dashboard( https://supabase.com/dashboard/project/dfmeqrofzpfmickppnmd/auth/url-configuration ) → Authentication → URL Configuration
      2. В поле "Site URL" укажите: https://nextgen.appletownworld.com/admin/callback
     
   - Google Analytics 4: 
     Получите Measurement ID и обновите переменную NEXT_PUBLIC_GA4_ID в настройках проекта:
     https://vercel.com/nextgen-9c28d5d1/nextgen/settings/environment-variables

   - Facebook Pixel:
     Получите Pixel ID и обновите переменную NEXT_PUBLIC_FACEBOOK_PIXEL_ID

   - Resend:
     Создайте API ключ на https://resend.com/api-keys и обновите переменную RESEND_API_KEY

После настройки всех ключей:
- Отслеживание конверсий будет работать автоматически (необходимо подождать 24 часа для корректной работы Google Analytics)
- Аналитика посещений будет доступна в соответствующих дашбордах
- Сайт будет полностью готов к работе на финальном домене
